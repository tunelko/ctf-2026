# Procedural Labyrinth — VishwaCTF 2026 (Crypto)

## TL;DR

Solve a 51x701 ASCII maze via BFS, then interpret each left/right zigzag as a binary bit (L=1, R=0) to decode the flag.

## Description

> A maze was generated by a classified program. Somewhere inside it, a message is waiting. The question is — how do you read it?

The challenge serves a text file containing an ASCII maze with metadata:

```
# Seed      : 0x1e522e9c
# Dims      : 51 x 701
# Entry     : col=0,    row=1      [ marked > ]
# Exit      : col=50,   row=699    [ marked < ]
# HINT: The map is just the canvas.
#       The true message is the journey you take to escape it.
```

## Analysis

The maze is a standard grid: `#` = wall, `.` = passage, `>` = entry, `<` = exit.

Solving via BFS reveals a path of length 1089 that zigzags downward. After the initial 33-step entry corridor, the path follows a strict pattern:

```
D D [R or L] D D [R or L] D D [R or L] ...
```

Every two rows, the path takes exactly one horizontal step — either Right or Left. This produces **344 single-bit decisions** (plus one special 13-step exit corridor at the end, which is discarded).

344 bits = 43 bytes. The hint ("the true message is the journey") confirms that these directional choices encode data.

## Vulnerability

**CWE-311: Missing Encryption of Sensitive Data** — The flag is embedded directly in the maze's solution path structure without any cryptographic protection. Each L/R turn encodes one bit of plaintext.

## Encoding

| Direction | Bit |
|-----------|-----|
| Left (L)  | 1   |
| Right (R) | 0   |

Bits are read MSB-first in groups of 8 to produce ASCII characters.

## Solve Script

```python
#!/usr/bin/env python3
from collections import deque

with open("index.html") as f:
    lines = [l.rstrip('\n') for l in f]

maze = [l for l in lines if len(l) == 51 and all(c in '#.><' for c in l)]
rows, cols = len(maze), 51

start = end = None
for r, row in enumerate(maze):
    for c, ch in enumerate(row):
        if ch == '>': start = (r, c)
        if ch == '<': end = (r, c)

# BFS
parent = {start: None}
queue = deque([start])
while queue:
    r, c = queue.popleft()
    if (r, c) == end: break
    for dr, dc in [(0,1),(1,0),(0,-1),(-1,0)]:
        nr, nc = r+dr, c+dc
        if 0 <= nr < rows and 0 <= nc < cols and maze[nr][nc] != '#' and (nr,nc) not in parent:
            parent[(nr, nc)] = (r, c)
            queue.append((nr, nc))

# Reconstruct path
path = []
node = end
while node:
    path.append(node)
    node = parent.get(node)
path.reverse()

# Direction sequence
dirs = []
for i in range(1, len(path)):
    dr = path[i][0] - path[i-1][0]
    dc = path[i][1] - path[i-1][1]
    dirs.append({(0,1):'R',(1,0):'D',(0,-1):'L',(-1,0):'U'}[(dr,dc)])

# Skip entry corridor
skip = next(i for i, d in enumerate(dirs) if d != 'R')
rdirs = dirs[skip:]

# Parse zigzag segments: groups of D's followed by H moves
segments = []
i = 0
while i < len(rdirs):
    if rdirs[i] == 'D':
        j = i
        while j < len(rdirs) and rdirs[j] == 'D': j += 1
        k = j
        while k < len(rdirs) and rdirs[k] in ('R','L'): k += 1
        segments.append((''.join(rdirs[j:k]),))
        i = k
    else:
        i += 1

# Extract single-step bits only (skip exit corridor)
bits = [s[0] for s in segments if len(s[0]) == 1]  # 344 bits

# Decode: L=1, R=0, MSB first
bitstr = ''.join('0' if b == 'R' else '1' for b in bits)
flag = ''.join(chr(int(bitstr[i:i+8], 2)) for i in range(0, len(bitstr)-7, 8))
print(flag)
```

## Key Lessons

- Maze-based steganography: the solution path's directional choices can encode arbitrary binary data
- A 51-wide maze with 701 rows provides 344 L/R decisions = 43 bytes, enough for a flag
- BFS gives the unique shortest path, ensuring deterministic decoding
- The hint "the journey you take to escape it" directly points to the solution path as the data carrier
